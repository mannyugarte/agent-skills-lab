# Agent Skills Lab

A collection of skills for Claude Code and other AI agents.

## Skills

| Skill | Description |
|-------|-------------|
| [procreativ-contracts](./procreativ-contracts) | Generate tailored Schedule A and Schedule B documents for client agreements across fixed-scope, retainer, and hourly engagements |
| [project-instructions-setter](./project-instructions-setter) | Create custom project instructions for new client engagements through a guided interview process |
| [synthesize-my-ai-preferences](./synthesize-my-ai-preferences) | Maintain consistent AI personalization instructions across multiple tools from a single source of truth |
| [talk-to-camera-script-converter](./talk-to-camera-script-converter) | Convert transcribed voice memos into short-form talk-to-camera video scripts optimized for pause-and-cut recording |

## Installation

Each skill can be installed individually. Navigate to the skill folder and follow its README for installation instructions.

## Adding New Skills

Each skill lives in its own folder with:
- `SKILL.md` - The skill prompt and instructions
- `references/` - Supporting reference files (optional)
